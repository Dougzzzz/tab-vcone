Repositorio criado para acompanhar as aulas ministradas no curso.dev

Trata-se de uma api usando node para cadastro de usuários.
Atualmente estou nas aulas de criação do script para que o projeto suba com o docker-compose com apenas um comando.

Conhecimentos adiquiridos até o momento:

- Uso do Git;
- DNS
- Publicação do projeto usando a vercel
- Organização de tarefas usando GitHub Project
- Desenvolvimento orientado a testes
- Docker - princípios e uso do docker-compose
- Banco de dados
- Migrações
